NutriLens Gizlilik Politikası



NutriLens, kullanıcı gizliliğine önem verir. Uygulama kapsamında paylaşılan kişisel veriler, yalnızca hizmet sunumu ve uygulama deneyimini geliştirmek amacıyla işlenir.



Toplanan Veriler



NutriLens aşağıdaki verileri işleyebilir:



Kamera ile yüklenen yemek ve besin görselleri

Besin ve kalori analiz sonuçları

Kullanıcı tercihleri ve uygulama ayarları

Teknik kullanım verileri (çökme kayıtları, cihaz bilgileri vb.)

Verilerin Kullanım Amaçları



Toplanan veriler:



Besin analiz hizmeti sunmak,

Uygulama performansını artırmak,

amacıyla kullanılmaktadır.

Veri Paylaşımı



Kişisel veriler, yasal yükümlülükler dışında üçüncü kişilerle satılmaz veya paylaşılmaz.



Veri Güvenliği



NutriLens, verilerin korunması için makul teknik ve idari güvenlik önlemleri uygular.

NutriLens Privacy Policy



NutriLens values user privacy. Personal data shared through the application is processed solely for providing services and improving user experience.



Data Collected



NutriLens may process:



Food and meal images uploaded via camera

Nutrition and calorie analysis results

User preferences and app settings

Technical usage data (crash logs, device information, etc.)

Purpose of Processing



Collected data is used to:



Provide nutrition analysis services,

Improve user experience,

Enhance app performance,

Ensure security and error monitoring.

Data Sharing



Personal data is not sold or shared with third parties except where required by law.



Data Security



NutriLens applies reasonable technical and administrative measures to protect user data.



KVKK \& GDPR Compliance

Turkish KVKK (Law No. 6698),

EU GDPR regulations.
