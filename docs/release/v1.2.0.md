## Full changelog (since 1.1.0)

### Features
- **Product comparison tray** + one-tap branded share card from the compare screen.
- **Real healthier alternatives** — surfaces actual higher-scoring products in the same category instead of placeholders.
- **USDA FoodData Central** data source added to the barcode chain (broader nutrition coverage when Open Food Facts / community DB miss).
- **Higher-fidelity AI analysis** — images sent to the vision model bumped from 1600px → 2048px long edge for denser Turkish labels; packaged-product detection routes "this is a packaged product, scan its barcode" back to the scanner.

### Fixes
- Premium: log RevenueCat in on cold-start session restore.
- l10n: localized remaining hardcoded Turkish UI strings.
- Ads: never serve Google test ad units in release builds.
- History: surface scan-history sync errors + merge local/remote history so unsynced scans show.

### Backend / infra
- `gemini-proxy` edge function updated (live at v38) — image-quality + prompt changes.
- `USDA_FDC_API_KEY` wired into Codemagic build env.
- Supabase security/perf hardening migration (2026-06-21) verified: SECURITY DEFINER RPCs locked to `authenticated`, `auth.uid()` checks enforced, `grant_bonus_scan` capped at 3/day (no paywall bypass).

## Free trial — enablement (your action, no app release needed)

The paywall **auto-detects** a free trial and switches the CTA to
"X Gün Ücretsiz Dene" the moment one is configured — `_freeTrialDays()` in
[paywall_screen.dart](../../lib/features/premium/presentation/screens/paywall_screen.dart)
reads it from `introductoryPrice` (price 0) or the Google Play
`defaultOption.freePhase`. Nothing else to ship. To turn it on:

1. **Play Console** → Monetize → Subscriptions → your base plan → add an
   **Offer** with a *Free trial* phase (e.g. 3 or 7 days). Activate it.
2. **RevenueCat** → Offerings → make sure the package points at that base
   plan/offer. No code change.
3. Test on an internal-track build with a license-tester account → the
   paywall should show the trial badge + "X Gün Ücretsiz Dene".

> Savings % is computed live from the real monthly vs annual prices
> (`_annualSavingsPercent`) — the old hardcoded "%40" is gone, so it can
> never disagree with the store prices.
