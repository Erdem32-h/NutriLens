## Changelog (since 1.1.0)
**Features**
- Product comparison tray + branded share card.
- Real higher-scoring alternatives (no more placeholders).
- USDA FoodData Central added to the barcode chain.
- AI images 1600→2048px; packaged-product detection routes back to scanner.
- Premium meal sync to Supabase (free users stay local-only).
- Paywall redesign: per-month price, live savings %, trial-aware CTA, full l10n.

**Fixes** — RevenueCat login on cold-start restore; remaining TR strings localized; no test ad units in release; scan-history sync errors surfaced + local/remote merge.

**Backend** — `gemini-proxy` edge fn (live v38); `USDA_FDC_API_KEY` in Codemagic; 2026-06-21 security hardening verified (SECURITY DEFINER RPCs locked, `grant_bonus_scan` capped 3/day).

## Free trial (your action — no release needed)
Paywall auto-detects a trial via `introductoryPrice` / `defaultOption.freePhase` and flips the CTA to "X Gün Ücretsiz Dene".
1. Play Console → Subscriptions → base plan → add **Offer** → *Free trial* phase → **Activate**.
2. RevenueCat → Offering points at that base plan. No code change.
3. Test with a never-subscribed license-tester account.

Savings % is computed live (`_annualSavingsPercent`) — old hardcoded "%40" removed.
